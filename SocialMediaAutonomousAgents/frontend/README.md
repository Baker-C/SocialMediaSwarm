# Frontend dashboard

Operator UI for the Social Media Autonomous Agents backend.

**Documentation:** [`docs/PROJECT.md`](../../docs/PROJECT.md) → [frontend-dashboard](../../docs/subsystems/frontend-dashboard.md).

## Quick start

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000). Set **`REACT_APP_API_URL=http://localhost:8000`** in `.env` when pointing at the backend.

For Docker and the full stack, see [operations](../../docs/subsystems/operations.md). API contract: [api-and-dashboard](../../docs/subsystems/api-and-dashboard.md).
